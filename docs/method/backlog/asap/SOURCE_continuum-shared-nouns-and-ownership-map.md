# Shared Nouns And Ownership Map

## Why

The active stack still risks shadow ownership and synonym drift across Echo,
`git-warp`, `warp-ttd`, Wesley, and Continuum.

## Goal

Produce one small coordination artifact that answers:

- which repo owns each shared noun
- which repo publishes the canonical contract surface for that noun
- which repos are consumers only

## Seed Nouns

- worldline
- strand
- braid
- observer
- receipt
- witness
- effect emission
- observer trace
- delivery observation
- session mode

## Done When

- the map exists in Continuum
- each noun has one explicit owner
- shadow homes are called out as debt or retired
