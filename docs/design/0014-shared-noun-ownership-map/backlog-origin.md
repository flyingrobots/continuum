# Shared Noun Ownership Map

**Legend:** SOURCE

This cycle promotes `docs/method/backlog/asap/SOURCE_continuum-shared-nouns-and-ownership-map.md`.

That backlog note asked for one small coordination artifact answering:

- which repo owns each shared noun
- which repo publishes the canonical contract surface
- which repos are consumers only

The witness/merge/browser work since then made the right refinement obvious:
ownership has to be split across semantics, contract home, runtime truth, and
product surface.

This cycle freezes that law so downstream repos can stop inventing shadow
ownership.
