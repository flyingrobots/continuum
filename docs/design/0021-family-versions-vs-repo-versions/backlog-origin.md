# Backlog Origin

Origin discussion:

- The stack now spans Continuum, Wesley, Echo, `git-warp`, and `warp-ttd`.
- Some things must remain shared and versioned together, but repos should not
  be forced into one lockstep release number.
- Compatibility needs to become explicit and inspectable rather than ambient
  repo folklore.

Prompt distilled into this packet:

**Write down the release doctrine that versions shared semantic families
independently from repositories and keeps interoperability explicit.**
